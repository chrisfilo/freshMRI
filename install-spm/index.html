install SPM

May 28, 2014

SPM1 is the original gangster of open source neuroimaging software.

freshSPM is tailored for SPM12b, providing as much backward compatibility with SPM8 as possible. None of the freshMRI code is likely to work in older versions (though your old code may work in SPM8/SPM12b).

SPM is far and away the easiest package to install (assuming you already have MATLAB installed). Unlike most of the other tools on freshMRI, You can probably get away with just downloading through a browser, clicking on the zipped file to extract it, and dragging it to ~/Documents/MATLAB.

There are a number of reasons why this is not the best way to start a trek into the black heart of neuroimaging data analysis, but rather than digress further, let’s just take this as an opportunity to get in some practice with the command line.

To set up SPM freshMRI-style:

First, insall MATLAB (whoever provides your site license should have instructions for how to do this).

Go to the SPM12b download page, fill out the brief form, and click the link:

If your browser is configured to let you select a download location, choose:

Fire up the Terminal. <div class="window"> <nav class="control-window"> close minimize deactivate </nav> <h1 class="titleInside">Terminal</h1> <div class="container"><div class="terminal"><table><tr><td class='gutter'><pre class='line-numbers'>$ $ $</pre></td><td class='code'><pre>cd ~/Documents/MATLABunzip spm12b.zipmatlab &</pre></td></tr></table></div></div> </div>

To add spm12b to the path:

Check out the post on how to set the path in MATLAB for an in-depth graphical walkthrough. To add spm12b, use the ‘Add Folder’ button (NOT ‘Add with Subfolders’). spm does this automatically through its own functions, and it can confuse MATLAB if you choose ‘Add with Subfolders’

You can save the path with spm12b on it unless you have a reason not to (e.g., you’re working in another version of spm, you’re working with functions that conflict with spm’s, etc.).

Launch that sucker:

If all goes according to plan, hit the MATLAB Command Window2 with:
